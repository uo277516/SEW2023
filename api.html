Aplicación para calcular distancias con puntos

Introducir coordenadas

En el siguiente bloque puedes arrastrar un archivo con coordenadas. A partir de ellas, se creará un mapa con un marcador con tu ubicación, unido con una línea a las coordenadas proporcionadas. Una vez con las coordenadas en el mapa, con el botón "Generar Información" podrás descargar un archivo .txt el cual te informará de las distancias entre los puntos.
